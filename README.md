# RJay 記帳網站

> 一套純前端的個人收支記帳工具，具備預算追蹤與圖表分析，資料儲存於瀏覽器本機。
> 線上 Demo：https://expense-tracker-eosin-three-39.vercel.app/

## 專案簡介

一個不需要後端、開啟即用的記帳應用。使用原生 JavaScript 搭配 Chart.js，
實作收支記錄、月份管理、預算控管與視覺化分析，所有資料透過瀏覽器
localStorage 持久化，重新整理也不會遺失

## 功能

- **收支記錄**：新增交易（名稱、金額、分類、收入/支出），即時更新餘額
- **月份管理**：可切換不同月份檢視，資料依月份自動篩選統計
- **預算控管**：設定當月預算，進度條即時顯示使用比例，
  達 80% 提示「即將超支」、超過 100% 警示「已超出預算」
- **圖表分析**：
  - 支出比例（甜甜圈圖）：依分類統計當月支出佔比
  - 收支趨勢（折線圖）：呈現當月每筆交易的收支變化
- **紀錄管理**：單筆刪除、一鍵清除全部（含二次確認）
- **資料持久化**：localStorage 儲存，無需後端與資料庫
- **響應式設計**：桌機、平板、手機皆適配

## 實作重點

- **無框架的狀態管理**：不依賴 Vue/React，用原生 JS 維護交易陣列與
  畫面同步，練習手動處理資料流與 DOM 更新
- **依日期動態篩選**：所有統計（餘額、圖表、預算）都會依當前選擇的
  月份即時重算，而非全部混在一起
- **圖表生命週期處理**：切換月份或更新資料時，先 destroy 舊圖表
  再重繪，避免 Chart.js 實例重疊造成的殘影問題
- **玻璃擬態 UI**：以 backdrop-filter 打造毛玻璃卡片與漸層背景

## 本機執行

這是純靜態網站，不需要安裝任何套件：

1. 下載專案
2. 直接用瀏覽器開啟 `index.html` 即可使用

（或用 VS Code 的 Live Server 擴充功能啟動，體驗較佳）

## 說明

本專案為個人前端練習，資料僅儲存在使用者本機瀏覽器，不上傳任何伺服器
